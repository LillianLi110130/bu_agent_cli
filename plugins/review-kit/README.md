# review-kit

Built-in example plugin for the V2 plugin system.

Resources included:

- `@review-kit:code-review`
- `review-kit:reviewer`
- `/review-kit:review`
- `/review-kit:summarize`

Quick try:

```text
/plugins list
/plugins show review-kit
/review-kit:review auth flow
/review-kit:summarize src/auth.py
@review-kit:code-review Review the current branch changes
```
