# K+ 检索

当用户需要使用 K+ 知识库做检索时，优先使用这个技能。

## 已支持的知识库 service_id

- `serviceId: "123"`
  债券市场交易相关法律法规、业务规则、通知、问答
  外汇市场相关制度、办法、通知和监管口径
  托管、登记、结算、清算相关规则
  金监局、原银保监会、人行、交易所、中债登、上清所等机构发布的制度法规

## 使用方法

1. 先从用户问题中提炼出适合检索的关键词
2. 选择本次检索对应的知识库 `serviceId`
3. 运行工作区内的脚本 `skills/kplus-search/scripts/search.py`，通过参数 `--question` 传入检索问题，通过参数 `--serviceId` 传入对应的知识库 `serviceId`
4. 阅读返回结果，提炼与用户问题最相关的信息
5. 如果结果太宽泛，就把关键词改得更具体，再检索一次

## 回答要求

- 先直接回答用户问题
- 再列出命中的关键结果线索
- 如果命中的是法规制度内容，不要编造法规条文
- 如果证据不足，要明确说明并建议用户补充关键词
